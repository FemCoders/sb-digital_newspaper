# Digital Newspaper
Deberás desarrollar una sencilla API REST utilizando Spring Boot que permita la gestión de artículos en un periódico digital, tendrá una arquitectura MVC de 3 capas, estilo cliente servidor (tipo API Rest) con conexión a una base de datos en PostgreSQL. El resultado final será el testeo de endpoints con Postman.

Revisa el briefing del taller ya que será un ejecicio por partes.

### Instrucciones
Crea el proyecto con Spring Inicializr y añádelo a este repositorio de github. 

### Tecnologías
- Java 21
- Spring Boot 3
- PostgreSQL
